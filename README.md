# Human Helper: Transition Command Deck

Interactive frontend prototype for helping people thrive during the AI transition.

## Run locally

1. From this folder, start a static server:

```bash
python3 -m http.server 8080
```

2. Open:

```
http://localhost:8080
```

## What it includes

- Humane market constellation with all listed markets.
- Personal baseline controls (life systems, risk, time, horizon).
- Dynamic Thrive Score and weakest-system signal.
- Auto-generated two-week transition sprint plan.
- Scenario comparison across near, mid, and long horizons.
